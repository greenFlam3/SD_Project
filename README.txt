Como Executar o Projeto

1. Teste Completo

Para testar o projeto na sua totalidade, basta iniciar o RMIServer e, em seguida, o RMIClient.

2. Teste das Três Primeiras Funcionalidades

Caso queira testar apenas as três primeiras funcionalidades, execute os seguintes comandos na ordem correta:
	1 - Iniciar URLQueueServer;
	2 - Iniciar StorageBarrelServer;
	3 - Iniciar TestURLQueue;
	4 - Iniciar Downloader;
	5 - Iniciar TestSearch;
Notar que tanto o TestURLQueue como o TestSearch são códigos usados para testar as funcionalidades e, por isso, é que se encontram com, pelo menos, um URL adicionado.
No entanto, é possível não só utilizar outros URL's como também adicionar mais URL's.
